Gradiente em CSS

Os gradientes CSS permitem que você exiba transições suaves entre duas ou mais cores especificadas. Aplicamos folhas de estilo no documento HTML com a tag <style>. E surgiu a propriedade: background-image ‒ Imagem de fundo, que nesse caso servirá para aplicar o gradiente.

Greadiente Linear

Para criar um gradiente linear, devemos definir pelo menos duas cores.Também, definiremos um ponto inicial e uma direção (ou um ângulo) junto com o efeito de gradiente.

Gradiente Radial

Um gradiente radial é definido pelo seu centro.
