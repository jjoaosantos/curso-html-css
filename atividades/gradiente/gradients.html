Gradiente em CSS

Os gradientes CSS permitem que você exiba transições suaves entre duas ou mais cores especificadas. Aplicamos folhas de estilo no documento HTML com a tag <style>. E surgiu a propriedade: background-image ‒ Imagem de fundo, que nesse caso servirá para aplicar o gradiente.

Greadiente Linear

Para criar um gradiente linear, devemos definir pelo menos duas cores. Também, definiremos um ponto inicial e uma direção (ou um ângulo) junto com o efeito de gradiente.

background-image: linear-gradient(direção, cor1, cor2, ...);

Top to bottom (Padrão)

background-image: linear-gradient(blue, yellow);

Left to right

background-image: linear-gradient(to right, blue, yellow);

Diagonal

background-image: linear-gradient(to bottom right, blue, yellow);

Ângulos

  • 0 degree - Para cima
  • 90 degree - Para a direita
  • 180 degree Para baixo
  • 270 degree - Para a esquerda

Gradiente Radial

Um gradiente radial é definido pelo seu centro

background-image: radial-gradient(tamanho da forma na posição, cor inicial, ..., cor final);

Evenly Spaced (Padrão)

background-image: radial-gradient(red, yellow, green);

Differently spaced

background-image: radial-gradient(red 5%, yellow 15%, green 60%);

Set Shape

background-image: radial-gradient(circle, red, yellow, green);
